## Steps

1. `crates/yee-cli/src/main.rs`: change `run_validate_list(target)` →
   `run_validate_list(target, json: bool)`. Build the filtered
   `Vec<CaseDescriptor>` as today. If `json`, print
   `serde_json::to_string_pretty(&cases)` and return `ExitCode::SUCCESS` before
   the table-building code. Otherwise the existing table path, unchanged.
2. Update the `Command::Validate { target, json, list }` arm to call
   `run_validate_list(target, json)`.
3. Update the crate-level `//! yee validate …` synopsis and the `Validate`
   command doc-comment to mention `--list --json` (JSON inventory, no solver run).
4. `crates/yee-cli/tests/cli_validate.rs`: add `yee_validate_list_json_runs`
   (NOT `#[ignore]`'d) — run `yee validate --list --json`, assert exit 0,
   `stdout.trim()` starts with `[` and ends with `]`, and contains `"mom-001"`,
   `"fem-eig-006"`, `"SkippedGateOpen"`, `"Run"`.
